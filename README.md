# vue-laravel-MPA testing template

## Required
+ Node.js
+ NPM
+ Composer

### Installation
```
git clone https://github.com/zzharuk/vuelara-template-mpa.git
```
```
cd vuelara-template-mpa
```
```
npm i
```
```
composer install
```
### PHP server
```
php artisan serve
```
### Fron-end VUE watch
```
npm run watch
```
